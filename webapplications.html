Notepad

Advantages

Free Software

Provides good basic grounding in coding

Good editor for creating HTML documents

Disadvantages

Very Basic Program

Can't add an image to a document

Can't change font size or colour

Notepad ++

Advantages

Free open source Software

Maximum functionality through menu bar at the top

Can have many windows open at the same time

Syntax Highlighting - Code errors

Disadvantages

User Interface - looks like Windows 95

Not Cross Platform - no version for Linux

Plugins failure to install

Dreamweaver

Advantages

Autofill - it suggests what the coder should do with code

Split screen shows updated design after code is changed

Highlights mistakes so they can be quickly corrected

Can find and replace items in your code easily

Disadvantages

Can be intimidating for a beginner

Not Browser based - code can look different when interpreted by the browser

Not suited for beginners/learners as the program does the coding

GoDaddy

Advantages

Ideal for newcomers and small business - takes your website from design to registration and hosting

Large selection of web hosting and website plans

Find customers with built-in Search Engine Optimization and Email Marketing tools

Disadvantages

One of the largest in the business - customer service inconsistent

Demands minimum 3 year commitment on some plans

Others such as Bluehost or GreenGeeks may be better option for small business

Wordpress

Advantages

Free open source Content Management system

Over 45,000 plugins available so little or no code has to be written

Little or no technical knowledge required to edit pages and update content

One website is all that is needed - website will adapt to desktop, tablet or smart phone formats

Disadvantages

Updating content, while easy to do, can interfere with functionality

As it is opensource, it is vulnerable to hacking

Frequent updates can cause compatibility issues
